# habical

Трекер задач & календарь & друзья

## Структура проекта
- `app/` — корневая настройка приложения, MaterialApp, тема, root screen
- `cubits/` — cubit’ы и state’ы
- `models/` — сущности
- `repositories/` — работа с данными
- `services/` — логика вроде прогресса и генерации task из habit
- `screens/` — экраны
- `widgets/` — общие переиспользуемые виджеты
